# Text Snap

A simple Vertical Snap Text example.

[See here an example](https://codepen.io/molavec/pen/wvmqEyG)


## File structure

```
/-- js
/-- sass
/index.html
```

## How to
Start project and open your browser http://localhost:1234
```
yarn start
```

Public code in `./dist` directory
```
yarn build
```
